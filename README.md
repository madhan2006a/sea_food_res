# அலைகடல் உணவகம் (Alai Kadal) - Premium Coastal Dining

Authentic Tamil coastal cuisine crafted with passion and served with warmth.

## About the Project
This is the official website for **அலைகடல் உணவகம்**, featuring a modern, premium design inspired by the ocean.

## Tech Stack
- **Framework**: React with Vite
- **Styling**: Tailwind CSS & Lucide Icons
- **Components**: shadcn/ui
- **Type Safety**: TypeScript

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Production Build**:
   ```bash
   npm run build
   ```
